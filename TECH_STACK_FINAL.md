## ✅ 最终确定的技术栈

### 前端
```
React 18 + Vite
├─ UI框架：自己写组件（不用组件库）
├─ CSS方案：Tailwind CSS
├─ 字体：Google Fonts（Noto Serif SC + Ma Shan Zheng）
├─ 状态管理：React内置useState（暂不需要复杂状态管理）
└─ 路由：后期添加（先用单页面验证）
```

**选择理由**：
- ✅ React生态最大，AI生成代码质量好
- ✅ Vite启动快，开发体验好
- ✅ 不用组件库：完全控制中国风样式
- ✅ Tailwind CSS：快速开发 + 易于定制中国风主题
- ✅ 简单即美：只写需要的组件

---

### 后端（第二阶段）
```
FastAPI
├─ 数据库：SQLite（零配置）
├─ 数据源：akshare
└─ AI：DeepSeek API
```

**选择理由**：
- ✅ 现代化、类型安全
- ✅ 性能好、自动文档
- ✅ 与Python数据科学生态配合好
- ✅ SQLite适合个人项目

---

### 部署（第三阶段）
```
Docker + Docker Compose
```

**选择理由**：
- ✅ 环境一致
- ✅ 部署简单
- ✅ 用户已确认
