# 股票分析系统 - 界面设计文档（MVP版本）

## 文档信息

**创建日期**：2026-01-27
**版本**：v1.0
**设计阶段**：MVP原型设计
**页面数量**：4个

---

## 一、设计概述

### 1.1 设计目标

创建一个简洁、专业、易用的股票分析系统界面，支持：
- 漏斗型选股（3套策略）
- 选股结果展示
- 个股深度分析（DCBAEF六维度）
- 历史记录回溯

### 1.2 设计原则

- **简洁优先**：信息层次清晰，不堆砌
- **数据可视化**：用表格、图表、颜色提升可读性
- **响应式设计**：支持不同屏幕尺寸
- **快速加载**：静态页面，轻量级

### 1.3 色彩方案

```
主色调：
- 主色：#2563eb（蓝色）- 专业、可信
- 辅助色：#64748b（灰色）- 中性信息

状态色：
- 绿灯：#22c55e（绿色）- 建议/优秀
- 黄灯：#eab308（黄色）- 谨慎/一般
- 红灯：#ef4444（红色）- 危险/避免
- 蓝色：#3b82f6（蓝色）- 信息/链接

背景色：
- 页面背景：#f8fafc（浅灰）
- 卡片背景：#ffffff（白色）
```

- 跳转到结果页

### 3.3 表格设计

**列说明**：
- 股票代码：6位代码
- 股票名称：中文名称
- 股价：当前价格（元）
- PE：市盈率
- ROE：净资产收益率（%）
- 评分：综合评分（0-10分），带颜色标识
  - 9-10分：🟢 深绿色
  - 7-8分：🟢 浅绿色
  - 5-6分：🟡 黄色
  - <5分：🔴 红色
- 操作：[详情]按钮

**排序功能**：
- 点击列头可排序
- 默认按评分降序

**分页功能**：
- 每页显示20条
- 支持翻页

## 九、设计风格更新（v1.1） - 中国山水画风格

**更新日期**：2026-01-27
**更新内容**：根据用户偏好，增加中国山水画风格设计选项

### 9.1 设计理念

将中国传统山水画美学与现代金融数据分析相结合，创造出独特的"水墨金融"视觉风格。

**核心理念**：
- **留白**：适当留白，避免信息过载
- **意境**：用色彩和布局传达宁静、沉稳的投资心态
- **雅致**：去繁就简，追求优雅简约
- **功能性优先**：保持金融产品的专业性，中国风作为点缀

---

### 9.2 双主题设计

#### 日间模式：水墨淡彩

**色彩方案**：
```
背景色：
- 宣纸白 #f5f2eb      - 主背景
- 淡宣纸 #faf8f3      - 次背景
- 纯白 #ffffff        - 卡片背景

文字色：
- 浓墨 #1a1a1a        - 主文字
- 淡墨 #4a4a4a        - 次文字
- 极淡墨 #7a7a7a      - 辅助文字

点缀色：
- 竹青 #2d5a3d        - 主色（按钮、链接）
- 淡竹青 #4a7c5f      - 悬停状态
- 赭石 #8b7355        - 警告色
- 朱砂红 #8b3a3a      - 危险色
- 靛蓝 #4a6fa5        - 信息色
```

**特点**：
- 模仿宣纸质感
- 水墨色调为主
- 清淡雅致
- 适合日间使用

---

#### 夜间模式：月夜山水

**色彩方案**：
```
背景色：
- 深黛蓝 #1a2329      - 主背景
- 墨色 #242d35        - 次背景
- 半透明墨色          - 卡片背景

文字色：
- 月光白 #e8dcc4      - 主文字
- 淡月光 #b8a88a      - 次文字
- 远山灰 #787060      - 辅助文字

点缀色：
- 月下青竹 #7fb069    - 主色
- 淡青竹 #9fc58f      - 悬停状态
- 金月 #d4a574        - 警告色
- 暗红 #c96f6f        - 危险色
- 月蓝 #7a9bc0        - 信息色
```

**特点**：
- 模仿月夜山水的意境
- 深黛蓝背景，温暖月光色调
- 护眼、柔和
- 适合夜间使用

---

### 9.3 字体系统

**字体优先级**：
```css
font-family: "STSong", "SimSun", "FangSong", "KaiTi", serif;
```

**字体用途**：
- **宋体（STSong）**：正文，端庄典雅
- **仿宋（FangSong）**：注释、说明
- **楷体（KaiTi）**：标题、强调

---

### 9.4 装饰元素

#### 基础装饰
1. **水墨印章**：右下角"股票分析"印章

3. **边框装饰**：淡墨色细边框
4. **水墨晕染**：卡片角落的淡淡墨迹效果

#### 文化点缀
1. **诗词**：
   - 首页："抽丝剥茧寻良骥，沙里淘金得如意"
   - 底部：投资风险提示诗句

2. **中文数字序列**：
   ```
   壹 · 选择筛选策略
   贰 · 微调筛选条件
   叁 · 开始筛选
   ```

3. **书法元素**：
   - 标题使用楷体
   - 重要文字使用竖排

### 9.5 主题切换功能

#### 自动切换
- 根据系统时间自动推荐主题
- 6:00-18:00：日间模式
- 18:00-6:00：夜间模式

#### 手动切换
- 用户可随时手动切换主题
- 切换时平滑过渡动画
- 保存用户偏好到本地存储

#### 技术实现
- 使用CSS变量（`data-theme`属性）
- LocalStorage保存用户偏好
- 平滑过渡动画（0.3s ease）

- **印章盖印**：重要操作时的印章动画

#### 导航效果

- **颜色过渡**：hover时淡墨色转为竹青色

#### 卡片效果

- **边框高亮**：选中时左侧竹青色边框

#### 原则
- **准确性优先**：图表必须清晰准确
- **适度中国风**：在周边元素体现中国风
- **避免过度装饰**：不影响数据可读性

#### 实现方案
- **图表本体**：保持现代风格（ECharts / Chart.js）
- **图表容器**：中国风边框、标题
- **图例**：使用中国风颜色
- **提示框**：水墨风格卡片
